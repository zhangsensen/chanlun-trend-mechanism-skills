# v0.1.0 发布说明

发布日期：2026-05-15

## 摘要

`v0.1.0` 是缠论趋势机制 Codex Skill 的首个独立公开版本。
本版本把 skill 合同、可复用脚本、参考文档、测试和发布文档整理到一个独立仓库中。

## 包含内容

- `chan-trend-mechanism` skill 合同。
- 单标的结构分析器。
- 固定 `12/26/9` 参数的 MACD 补列工具。
- 多标的结构面板工具。
- 数据格式说明。
- 机制与用语说明。
- 覆盖结构分析器、MACD 补列和面板行为的 synthetic test suite。
- 公开 README、系统文档、发布检查清单、更新日志和 MIT License。
- 2026-05-15 上证指数人类可读输出示例。
- harness 说明：模型只组织 canonical JSON 字段，不作为判断源。

## 输出合同

- 结构输出：`chanlun_structure_v0.2`
- 面板输出：`chanlun_panel_v0.1`
- JSON 是 canonical。
- Markdown 是确定性展示。

## 边界

- 只读取本地 closed-bar parquet 数据。
- 不使用会话文件、截图、外部接口或凭据。
- 不读取回测或收益评估产物。
- MACD 只作为辅助动能背景。
- 缺失 K 线只进入 warnings 和 unknowns，不推断补齐。
- 生成结论只保留结构观察，不输出动作指令。
